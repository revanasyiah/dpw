Uraian AJAX & jQuery + Contoh

AJAX

Asynchronous JavaScript and XML — teknik mengambil data dari server tanpa reload halaman.

jQuery

Library JavaScript untuk mempermudah manipulasi DOM. "Write less, do more."

Contoh kode:

// Ubah teks
$('#judul').text('Halo Dunia!');

// Sembunyikan
$('.kotak').hide();

// AJAX
$.ajax({
  url: 'https://api.example.com/data',
  success: function(data){
    $('#hasil').html(data);
  }
});
